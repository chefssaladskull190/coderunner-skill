## 🧪 Test case tips
Good test cases help grading work well. Use:

- One normal case
- One edge case
- One case with large input
- One case with small input
- One case that checks wrong assumptions

Write tests that match the task rules exactly. Small mistakes in test data can lead to bad grading results.

4. Select the exported file
5. Upload it
6. Check that the question looks right
